== Description ==

Plugin WordPress per la gestione e creazione di landing page personalizzate **senza dover usare Elementor o Gutenberg**.

== Installation ==

1. Esegui `composer install` nella cartella del plugin
2. Vai in WordPress → Plugin e attiva il plugin
3. Vai in WordPress → Landing Pages per creare la prima landing page
